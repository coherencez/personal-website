New Project

Built with bash scripts

A simple bash script to assist teammates in getting up and running on a new static web project. This was a precursor project to The NSS Yeoman Generator. Automatically creates file structure, and necessary files, and initializes a git repository. Includes fail-safe which prints the current directory and asks for user confirmation before continuing on. Later versions include Gulp, and Jasmine installs
